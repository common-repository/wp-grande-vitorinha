Use easily the font "Grande Vitorinha" in your website.

== Description ==

Grande Vitorinha font is inspired by beautiful cities of Espírito Santo in Brazil.
If you need know more about this font, check out the [official website](http://www.grandevitorinha.com.br/download.html).

== Installation ==

1. Upload `wp-grande-vitorinha.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Use the shortcode [wpgv][/wpgv]

== Frequently Asked Questions ==

= How change the font size? =

Use the attribute "size" in shortcode, e.g.: [wpgv size=70]WP[/wpgv]. The size of font use pixel and the default value is 60.

= How change the font color? =

Use the attribute "color" in shortcode, e.g.: [wpgv color="#bdc3c7"]WP[/wpgv]. You need to use the hexadecimal format and the default value is #333333.
